# Kölcsönző alkalmazás

### Technológiák
- Fejlesztés:
    - ui
        - angular 19
    - backend
        - java 21, spring
    - alkalmazás
        - egy spring-es projekt ami az angular-t is szolgáltatja és a backendet is

### Szolgáltatások
- Felület
    - Bárki
        - Bemutakozás megtekintése
        - Gépek listázása és keresése
        - Regisztráció
    - Felhasználó
        - Belépés
        - Foglalás egy gépre
        - Saját foglalások megtekintése
    - Adminisztrátor
        - Belépés
        - Gépek megtekintése/felvitele/módosítása/törlése
        - Foglalások megtekintése
